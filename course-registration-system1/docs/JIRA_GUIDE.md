# Jira Guide (Process Requirement)

The project spec requires Jira usage for task creation, assignment, progress tracking, and (optionally) sprint management.

## Suggested Jira setup
- Project type: Scrum (or Kanban)
- Epics:
  - Authentication & Roles
  - Courses (CRUD)
  - Terms / Sections
  - Student Enrollment
  - Admin Management
  - UI/UX
  - Security & Validation
  - Documentation

## Suggested workflow columns
- Backlog
- To Do
- In Progress
- In Review
- Done

## What to show in the demo
- Each team member has assigned tasks
- Tickets linked to commits/PRs (if you use GitHub integration)
- A clear history of progress
